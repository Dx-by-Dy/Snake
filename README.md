# Snake

## Описание
Классическая змейка, написанная на языке С++ в Visual Studio 2019, в которой вы управляете змейкой, поедая яблоки и увеличиваясь в длине.
Вы умираете при столкновении со стенами или с самим собой. 

## Управление
Используйте стрелочки.

## Системные требования
Windows XP и выше.

10 MB оперативной памяти.

Процессор Intel Pentium и выше.
